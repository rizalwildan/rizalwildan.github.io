Golang Tutorial Part 1: Pengenalan dan Instalasi

2020 / 05 / 16Tutorial Golang

Ini adalah tutorial pertama dalam seri tutorial Golang. Tutorial ini memberikan pengantar untuk Go. Kita juga akan belajar bagaimana menginstall Go di Mac OS, Windows, dan Linux.

Pengenalan

Go juga dikenal sebagai Golang adalah sebuah project Open Source, jenis bahasa pemrograman static typed yang dikembangkan oleh Google. Go mulai tersedia untuk umum pada bulan November 2009.

Go adalah bahasa pemrograman denagn sintaksis sederhana dan didukung oleh standard library yang sangat kuat. Go adalah bahasa yang powerful untuk membangun berbagai macam aplikasi. Mulai dari aplikasi command line interface, distributed microsystem dan bahkan cloud platform.

Instalasi

Go dapat diinstal pada ketiga platform Mac, Windows, Linux. Versi Go terbaru pada saat penulisan ini adalah go 1.14.3 kalian dapat mengunduh file binary sesuai dengan paltform machine kalian di sini https://golang.org/dl/

  • Mac Os

    Download Mac Os installer dari https://golang.org/dl/. Double tap untuk memulai installasi. Ikuti petunjuknya Golang akan terinstall di direktori /usr/local/go dan akan menambahkan folder /usr/local/go/bin ke PATH environtment variable kalian.

  • Windows

    Download MSI installer dari https://golang.org/dl/. Double tap untuk memulai installasi dan ikuti petunjuknya. Golang akan terinstall di direktori C:\Go dan akan menambahkan direktori C:\Go\bin ke PATH environtment variable

  • Linux

    Download file tar dari https://golang.org/dl/ unzip file tersebut ke direktori /usr/local.

    Tambahkan /usr/local/go/bin ke PATH environtment variable.

Untuk memastikan proses instalasi Go berhasil, ketikkan perintah go version di terminal.

Jika proses instalasi berhasil maka akan memunculkan versi Go yang telah terinstal. Berikut contoh outputnya.

$ go version
go verison go1.14.3 linux/amd64

Pada step ini kita telah berhasil menginstal Golang dengan versi 1.14.3.

Untuk tutorial selanjutnya kita akan melakukan setup project di Golang dan sedikit membahas go modules.

Happy Coding :)
